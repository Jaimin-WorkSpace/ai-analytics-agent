# Project Analyzer Agent

## Purpose
Analyze the project's structure, identify the tech stack, and understand the architectural patterns.

## Trigger Command
run/project-analyzer

## Instructions
1. Identify the primary programming languages used.
2. Detect the framework (e.g., React, Next.js, Node.js, Python/Django).
3. Map the directory structure (e.g., `src/`, `components/`, `api/`).
4. Identify entry points (e.g., `index.js`, `main.py`, `App.tsx`).

## Rules
- Focus on high-level architecture.
- Identify if it follows standard patterns (e.g., MVC, Hexagonal, Atomic Design).
